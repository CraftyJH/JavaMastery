📚 Variables and Data Types

In Java, every variable has a specific type that determines its size and layout in memory. Java has two categories of data types: primitive types (int, double, boolean, etc.) and reference types (objects, arrays, strings).

// Primitive data types
int age = 25;
double price = 19.99;
boolean isActive = true;
char grade = 'A';

// Reference types
String name = "JavaMastery";
Integer wrappedAge = new Integer(age);

🔄 Control Flow Statements

Control flow statements allow you to control the execution order of your program. Java supports if-else, switch, and loop constructs (for, while, do-while).

// If-else statement
int score = 85;
if (score >= 90) {
    System.out.println("Grade: A");
} else if (score >= 80) {
    System.out.println("Grade: B");
} else {
    System.out.println("Grade: C");
}

// For loop
for (int i = 0; i < 5; i++) {
    System.out.println("Count: " + i);
}

📦 Methods and Scope

Methods are blocks of code that perform specific tasks. They help organize code and promote reusability. Understanding scope is crucial - variables declared in a method are local to that method.

// Method with parameters and return type
public static int calculateSum(int a, int b) {
    int sum = a + b;
    return sum;
}

// Method overloading
public static int calculateSum(int a, int b, int c) {
    return a + b + c;
}

// Main method
public static void main(String[] args) {
    int result = calculateSum(5, 3);
    System.out.println("Sum: " + result);
}

🏗️ Classes and Objects

A class is a blueprint for creating objects. An object is an instance of a class. Classes contain fields (attributes) and methods (behaviors).

// Defining a class
public class Car {
    // Fields (attributes)
    private String brand;
    private String model;
    private int year;
    
    // Constructor
    public Car(String brand, String model, int year) {
        this.brand = brand;
        this.model = model;
        this.year = year;
    }
    
    // Method
    public void start() {
        System.out.println("The " + brand + " " + model + " is starting!");
    }
}

// Creating objects
Car myCar = new Car("Tesla", "Model 3", 2024);
myCar.start();

🔄 Inheritance and Polymorphism

Inheritance allows a class to inherit properties and methods from another class. Polymorphism allows objects to take on many forms - a subclass can be treated as its parent class.

// Parent class
public class Animal {
    public void makeSound() {
        System.out.println("Some sound");
    }
}

// Child class
public class Dog extends Animal {
    @Override
    public void makeSound() {
        System.out.println("Woof! Woof!");
    }
}

// Polymorphism in action
Animal myPet = new Dog();
myPet.makeSound(); // Outputs: Woof! Woof!

🔐 Encapsulation and Interfaces

Encapsulation bundles data and methods that operate on that data within a class, restricting direct access to some components. Interfaces define contracts that classes can implement.

// Encapsulation with private fields and public getters/setters
public class BankAccount {
    private double balance; // Encapsulated field
    
    public double getBalance() {
        return balance;
    }
    
    public void setBalance(double balance) {
        if (balance >= 0) {
            this.balance = balance;
        }
    }
}

// Interface definition
public interface Drawable {
    void draw();
}

// Implementing an interface
public class Circle implements Drawable {
    @Override
    public void draw() {
        System.out.println("Drawing a circle");
    }
}

🔍 Searching and Sorting Algorithms

Understanding algorithms is crucial for efficient programming. Binary search finds elements in O(log n) time on sorted arrays. Merge sort and quick sort are efficient sorting algorithms.

// Binary Search
public static int binarySearch(int[] arr, int target) {
    int left = 0;
    int right = arr.length - 1;
    
    while (left <= right) {
        int mid = left + (right - left) / 2;
        
        if (arr[mid] == target) return mid;
        if (arr[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return -1;
}

🌳 Trees and Graphs Basics

Trees are hierarchical structures with nodes connected by edges. Graphs are more flexible structures where nodes can have multiple connections. Binary trees and binary search trees are commonly used in Java.

// Binary Tree Node
class TreeNode {
    TreeNode left, right;
    int value;
    
    TreeNode(int value) {
        this.value = value;
    }
}

// Insert into BST
public TreeNode insert(TreeNode root, int value) {
    if (root == null) return new TreeNode(value);
    if (value < root.value) 
        root.left = insert(root.left, value);
    else 
        root.right = insert(root.right, value);
    return root;
}

🌱 Introduction to Spring Framework

The Spring Framework provides comprehensive infrastructure support for building enterprise Java applications. It promotes loose coupling through Dependency Injection (DI) and Inversion of Control (IoC).

// Spring Bean Configuration
@Component
public class UserService {
    private UserRepository userRepository;
    
    @Autowired
    public UserService(UserRepository userRepository) {
        this.userRepository = userRepository;
    }
    
    public User findUserById(Long id) {
        return userRepository.findById(id);
    }
}

// Application.properties (Spring Boot)
# Server configuration
server.port=8080

🌐 Building REST APIs with Spring Boot

REST APIs are the backbone of modern web applications. Spring Boot makes it easy to create RESTful services using @RestController, @GetMapping, @PostMapping, and other annotations.

@RestController
@RequestMapping("/api/users")
public class UserController {
    
    @GetMapping
    public List<User> getAllUsers() {
        return userService.findAll();
    }
    
    @GetMapping("/{id}")
    public User getUserById(@PathVariable Long id) {
        return userService.findById(id);
    }
    
    @PostMapping
    public User createUser(@RequestBody User user) {
        return userService.save(user);
    }
}

🗄️ Database Integration with JPA/Hibernate

JPA (Java Persistence API) and Hibernate provide object-relational mapping (ORM) capabilities. They allow you to work with databases using Java objects instead of SQL.

@Entity
@Table(name = "users")
public class User {
    
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;
    
    @Column(nullable = false)
    private String name;
    
    @Column(unique = true)
    private String email;
    
    // Getters and Setters
}

@Repository
public interface UserRepository extends JpaRepository<User, Long> {
    Optional<User> findByEmail(String email);
}
